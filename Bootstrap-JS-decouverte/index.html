Rappels sur HTML et CSS ↑

HTML (HyperText Markup Language) est un langage de balisage utilisé pour structurer le contenu d'une page web. Il est composé de balises qui définissent la structure et le contenu de la page.
La structure d'une page HTML à la forme d'un arbre nommé DOM (Document Object Model). Chaque élément HTML est un nœud de l'arbre, et peut contenir d'autres éléments ou du texte.

Cette structure est ensuite stylisée à l'aide de CSS, qui permet de définir des règles de style pour les éléments HTML, comme la couleur, la taille, la police, la marge, le padding, etc.

Explications sur Bootstrap 5 ↑

Bootstrap est un framework front-end. Un framework est un ensemble de composants réutilisables et de conventions qui facilitent le développement d'applications web en fournissant une structure de base et des outils prêts à l'emploi. Bootstrap inclut des styles CSS, des composants JavaScript, et des modèles de mise en page qui permettent de créer des interfaces utilisateur cohérentes et réactives.

Voici quelques caractéristiques qui font de Bootstrap un framework populaire :

  • Composants réutilisables : Bootstrap fournit une vaste gamme de composants UI prêts à l'emploi, tels que des boutons, des formulaires, des cartes, des alertes, des modales, etc. Ces composants sont déjà stylisés et peuvent être utilisés sans effort.
  • Responsive design : Bootstrap est conçu pour être responsive dès le départ. Grâce à son système de grille et à ses classes utilitaires, il est facile de créer des mises en page qui s'adaptent automatiquement à différentes tailles d'écran, offrant ainsi une expérience utilisateur optimale sur mobile, tablette et desktop.
  • Styles CSS prédéfinis : Bootstrap offre des styles CSS pour la typographie, les tableaux, les images, et bien plus encore, ce qui permet de maintenir une apparence cohérente et professionnelle.
  • Plugins JavaScript : Bootstrap inclut de nombreuses extensions JavaScript pour ajouter des fonctionnalités interactives, comme les carrousels, les modales, les tooltips, les menus déroulants, et les accordéons, sans écrire de JavaScript complexe.
  • Personnalisation : Bootstrap peut être personnalisé en modifiant les variables Sass pour adapter les styles aux besoins spécifiques d'un projet.
  • Compatibilité multi-navigateurs : Bootstrap assure une compatibilité avec la plupart des navigateurs modernes, ce qui réduit les efforts nécessaires pour corriger des bugs d’affichage.

Pour utiliser Bootstrap, il suffit d'inclure les fichiers CSS et JavaScript de la bibliothèque dans votre page HTML, et d'ajouter les classes Bootstrap aux éléments que vous souhaitez styliser.
Pour inclure Bootstrap dans votre projet, vous pouvez utiliser les CDN (Content Delivery Network) fournis par Bootstrap, ou télécharger les fichiers directement sur le site officiel. Dans ce cas, vous devrez les ajouter à votre projet en les important dans votre fichier HTML.

Voici un exemple d'inclusion des fichiers Bootstrap via les CDN :

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC" crossorigin="anonymous">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-MrcW6ZMFYlzcLA8Nl+NtUVF0sA7MsXsP1UyJoMp4YLEuNSfAP+JcXn/tWtIaxVXM" crossorigin="anonymous"></script>

Pour plus d'informations, consultez la documentation officielle de Bootstrap.

Contenus réactifs ↑

Les contenus réactifs (en anglais, responsive content) sont des éléments d'interface utilisateur qui s'adaptent automatiquement à la taille de l'écran sur lequel ils sont affichés. Ils sont souvent utilisés pour afficher des informations de manière optimale sur les appareils mobiles, les tablettes et les ordinateurs de bureau.

Bootstrap propose des classes utilitaires pour masquer ou afficher des éléments en fonction de la taille de l'écran.
La classe d-none permet de masquer un élément sur tous les appareils, tandis que les classes d-block, d-sm-block, d-md-block, d-lg-block, et d-xl-block permettent d'afficher un élément uniquement sur les appareils de la taille spécifiée.
La documentation officielle de Bootstrap propose une liste complète des classes utilitaires pour les contenus réactifs : Documentation officielle de Bootstrap.

Bootstrap Icons est une bibliothèque d'icônes open-source conçue pour être utilisée avec Bootstrap. Elle contient une collection d'icônes vectorielles qui peuvent être facilement intégrées dans des projets web.

Menu déroulant Bootstrap ↑

Les menus déroulants Bootstrap sont des éléments d'interface utilisateur qui permettent d'afficher une liste d'options ou de liens lorsqu'un bouton ou un lien est cliqué. Ils sont souvent utilisés pour afficher des menus de navigation, des listes déroulantes, ou des options de sélection.

Les menus déroulants necessitent l'ajout de la librairie JavaScript Popper.js pour fonctionner correctement. Vous pouvez inclure Popper.js via CDN en ajoutant la ligne suivante dans votre fichier HTML :
<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.11.6/dist/umd/popper.min.js"></script>
